# **App Name**: Resin Revenue

## Core Features:

- Landing Page: Dynamic landing page for the 'Viviendo de Resina' course, highlighting key benefits and testimonials.
- Multi-Step Quiz: Interactive quiz to engage potential students and qualify them for the course.
- Personalized Result Display: Display customized result based on quiz answers, leading to course promotion.
- Testimonial Carousel: Showcase testimonials from successful students to build trust and credibility.
- Scarcity Timer: Implement a countdown timer to create a sense of urgency and encourage immediate action.
- Interactive Elements: Smooth animations and interactive elements enhance user experience and engagement.

## Style Guidelines:

- Primary color: White (#FFFFFF) for a clean and modern look.
- Secondary color: Purple (#9060E0) to convey sophistication and creativity.
- CTA button color: Green (#00FF00) for high visibility and a sense of action.
- Headline font: 'Playfair Display', a serif font for an elegant and classic look.
- Body font: 'Poppins', a sans-serif font for clean and readable text.
- Use Font Awesome icons related to crafting and resin art, integrating emojis for added emotional appeal.
- Mobile-first design with a full-width hero section and smooth transitions between quiz steps.
- Subtle animations such as fade-in and slide-up effects during quiz transitions and on the landing page.
